Completely disable comments and pingbacks site-wide and scrub them from the admin — in one click.

== Description ==

Comments invite spam and most sites don't need them. Disable Comments Clean
turns them off everywhere and removes every trace from the dashboard:

* Closes comments and pingbacks on posts, pages and custom post types.
* Hides existing comments from the front end.
* Removes the Comments admin menu, toolbar item and dashboard widget.
* Redirects the comments screen so it can't be reached.

Choose to disable everywhere, on posts only, or on pages only.

**Free features**

* Everything above.

**Pro features**

* Per-post-type granular control.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate — comments are off immediately.
3. Fine-tune under Settings → Disable Comments.

== Frequently Asked Questions ==

= Does it delete existing comments? =

No. It hides them from visitors but leaves your data intact, so you can
re-enable any time.
